# 🩺 DocAppoint - Doctor Appointment Manager

A modern Doctor Appointment Booking System built with **Next.js**, **MongoDB**, **Express.js**, and **Better Auth**. Users can browse doctors, book appointments, manage bookings, and securely authenticate using Google Login.

## ✨ Features

- 🔐 Google Authentication using Better Auth
- 👨‍⚕️ Browse all available doctors
- 📄 Doctor details page
- 📅 Book doctor appointments
- 📋 Dashboard to manage bookings
- ✏️ Update appointment information
- 🗑️ Delete appointments
- 📱 Fully responsive design
- 🎨 Modern UI with Tailwind CSS & Framer Motion
- 🔥 Toast notifications
- ⚡ Fast deployment with Vercel
